# Synth Terminal
*A cyberpunk roguelite TRPG system.*

Synth Terminal is an original tabletop RPG system inspired by roguelite
mechanics, cyberpunk worlds, and modular character growth.  
This repository serves as the development hub and public timestamp for the project.
